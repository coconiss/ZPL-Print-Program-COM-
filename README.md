# ZPL Universal RFID/Barcode Printer PRO 🖨️

이 프로젝트는 산업용 바코드 및 RFID 라벨 프린터(Zebra 등 ZPL 호환 프린터)를 제어하기 위한 강력하고 유연한 .NET Windows Forms 애플리케이션입니다. 

기존의 고정된 단일 바코드 인쇄 방식에서 벗어나, 사용자가 **직접 ZPL 템플릿과 변수를 설계하고 엑셀처럼 대량의 데이터를 일괄 인쇄(Batch Print)할 수 있는 동적 라벨링 시스템**을 제공합니다.

## ✨ 주요 기능 (Key Features)

### 1. 무한한 확장성을 가진 '동적 파라미터 시스템'
* ZPL 코드 내에 `{BARCODE}`, `{ITEM_NAME}`, `{PRICE}` 등 원하는 변수를 마음대로 정의할 수 있습니다.
* **엑셀(DataGrid) 스타일 입력:** 프로필에 정의된 변수들이 자동으로 그리드의 컬럼으로 생성됩니다. 엑셀 데이터를 복사/붙여넣기 하여 수백 장의 각기 다른 라벨을 한 번에 발행할 수 있습니다.

### 2. 스마트한 라벨 프로필 (Label Profile) 관리
* 라벨의 가로/세로 크기, 해상도(DPI), ZPL 템플릿, 사용할 변수 목록을 '프로필(Profile)' 단위로 저장하고 관리합니다.
* **변수 자동 추출:** 복잡한 ZPL 템플릿을 붙여넣고 버튼을 누르면, 템플릿 내부의 `{변수명}`을 프로그램이 자동으로 분석하여 입력 그리드에 추가합니다.
* **다중 템플릿 지원:** 바코드 텍스트(Human Readable)를 출력하는 템플릿과, RFID 인코딩만 수행하는 템플릿을 분리하여 저장/사용할 수 있습니다.

### 3. 데이터 참조 및 세그먼트(분할) 엔진
* **변수 참조:** 특정 변수에 다른 변수명(예: `BARCODE`)을 기본값으로 입력하면, 인쇄 시 해당 변수의 데이터를 자동으로 참조하여 가져옵니다.
* **세그먼트 포맷팅:** `3,4,4` 와 같은 규칙을 설정하면 연속된 문자열(예: `01012345678`)을 `010-1234-5678` 형태로 자동 분할하여 ZPL에 전달하고 UI에 즉시 시각화합니다.

### 4. 강력한 하드웨어 연동 및 자동화
* **다중 연결 지원:** COM (Serial), USB, TCP/IP (Ethernet) 연결을 완벽히 지원합니다.
* **COM 포트 상세 설정:** BaudRate, DataBits, Parity, StopBits를 UI에서 직접 제어할 수 있습니다.
* **스캐너 연동:** 빠른 스캔 입력창을 통해 바코드를 스캔하면 가장 첫 번째 파라미터로 자동 할당되며, 설정에 따라 **스캔 즉시 자동 인쇄**가 가능합니다.

### 5. 사용자 편의성 극대화
* **인쇄 미리보기:** 실제 프린터로 전송되기 전, 어떤 데이터가 매핑되었는지 팝업으로 검증할 수 있습니다.
* **설정 자동 저장:** 마지막으로 사용한 통신 포트, 선택한 프로필, 인쇄 옵션 등이 `appsettings.json`에 기록되어 프로그램 재시작 시 그대로 복원됩니다.

## 📂 프로젝트 아키텍처 (Structure)

UI 로직과 핵심 비즈니스 로직(Core)을 완벽히 분리하여 높은 유지보수성을 자랑합니다.

ZplPrinter/
├── Program.cs
├── Core/
│   ├── Models.cs             # 라벨 프로필, 동적 필드, 세그먼트 엔진 모델
│   ├── ProfileManager.cs     # JSON 기반 프로필 저장/로드 엔진
│   ├── PrinterService.cs     # COM/USB/TCP 원시(RAW) 인쇄 통신 모듈
│   └── SettingsManager.cs    # 앱 환경설정 자동 저장/복원 매니저
└── UI/
    ├── MainForm.cs           # 메인 데이터 그리드, 스캐너 연동 및 일괄 인쇄 화면
    ├── ProfileEditorForm.cs  # 프로필 파라미터 설계 및 ZPL 템플릿 편집 화면
    └── PrintPreviewForm.cs   # 인쇄 전 데이터 매핑 검증 화면
	
	
🚀 사용 방법 (How to Use)
연결 설정: 상단의 '연결 방식' 탭에서 프린터가 연결된 방식(COM, USB, TCP/IP)을 선택하고 올바른 포트나 IP를 입력합니다.

프로필 설계: [+ 새 프로필] 또는 [파라미터 편집] 버튼을 눌러 라벨 크기(mm)와 해상도(DPI)를 설정합니다.

ZPL 및 변수 세팅: 하단 텍스트 박스에 ZPL 코드를 입력하고 [ZPL에서 변수명 자동 추출] 버튼을 클릭하여 파라미터 그리드를 구성합니다. (필요시 세그먼트 규칙 추가)

데이터 입력: 메인 화면의 그리드에 데이터를 직접 타이핑하거나 엑셀에서 복사해 붙여넣습니다. 또는 '빠른 스캔 입력' 창을 활용합니다.

발행: [선택 인쇄] 또는 [전체 일괄 인쇄] 버튼을 눌러 작업을 완료합니다.

🛠️ 요구 사항 (Requirements)
Framework: .NET 8.0 (Windows Forms 기반)

OS: Windows 10 / 11

System.IO.Ports NuGet 패키지 (COM 통신용)
